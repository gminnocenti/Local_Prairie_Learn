What is Depth-First Search (DFS)?

Depth-First Search, or DFS, is a method for exploring a graph by going as deep as possible along a branch before backtracking. Imagine the graph as a collection of connected points, called "nodes." DFS starts from a chosen starting node and explores one path all the way to the end, moving from one node to the next until it reaches a point where it can't go further.

When DFS reaches the end of a branch or a node with no unvisited neighbors, it backtracks to the previous node and continues exploring the next branch. This process of diving deep and then backtracking continues until all nodes connected to the starting point have been visited.

By exploring as far along each path as possible before switching paths, DFS follows a depth-first approach, which is different from the level-by-level exploration of BFS.
